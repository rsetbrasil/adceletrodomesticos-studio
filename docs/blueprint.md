# **App Name**: CrediFacil

## Core Features:

- Product Catalog: Display a product catalog with categories, images, descriptions, prices, and availability.
- Shopping Cart: Implement a shopping cart to allow users to add products, update quantities, and view the order total.
- Customer Registration: Implement a customer information form during checkout (Name, CPF, Phone, Email, Address).
- Crediário Payment: Provide crediário (installment plan) as a payment option. Allow the user to define the number of installments.
- Order Finalization: Finalize the order, generate an order number, and display the installment plan details to the user.
- Admin notifications: Generate notifications and emails to the store admin after each purchase

## Style Guidelines:

- Primary color: Deep blue (#1A237E), representing trust and security, aligning with the financial aspect of the application.
- Background color: Very light blue (#E8EAF6), offering a clean and professional backdrop.
- Accent color: Purple (#7B1FA2), for highlighting important elements and CTAs, creating contrast without being too aggressive.
- Body and headline font: 'Inter' for a modern, machined, objective, neutral look. This font ensures readability and a professional feel.
- Use simple and clean icons to represent product categories, payment options, and other relevant actions.
- Design a clean and intuitive layout to help the users browse products, manage their shopping cart, and complete purchases efficiently.
- Implement subtle animations for button hover effects, transitions between pages, and cart updates, to enhance user experience without being distracting.
